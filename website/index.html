Local network monitoring

ScreensView показывает экраны Windows-компьютеров в локальной сети в одном Viewer.

Короткий showcase без документации на сайте: скачать Viewer, открыть репозиторий или перейти в README можно сразу с этой страницы.

Скачать Viewer README GitHub

Если прямой Viewer .exe не найден, кнопка откроет страницу последних релизов.

Живая стена мониторинга

Viewer собирает скриншоты нескольких Windows-машин и показывает их в одной сетке без отдельной облачной панели.

Удалённая установка и обновление

Агенты можно разворачивать и обслуживать из самого Viewer, включая выбор modern или legacy payload под целевую ОС.

Опциональная локальная проверка экрана

При необходимости Viewer может локально сверять скриншот с ожидаемым описанием экрана и помечать подозрительные состояния.

ScreensView остаётся небольшим self-hosted инструментом: подробная установка и эксплуатация описаны в README репозитория.
